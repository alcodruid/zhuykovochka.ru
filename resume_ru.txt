1. LibraryApp
   Роль: Android (Kotlin, MVVM)
   Описание: Android-приложение на Kotlin, реализующее каталог и управление библиотеками 
   с хранением данных в базе Room и загрузкой информации из внешнего JSON API.
   Ключевые особенности:
   - Полная реализация CRUD-функциональности для управления данными.
   - Успешная интеграция с внешним REST API для синхронизации данных.
   - Архитектура MVVM + Room для локального хранения.
   - Material Design UI с адаптивными компонентами.

2. TimeConverter
   Роль: Android (Kotlin)
   Описание: Мобильное приложение для конвертации единиц времени, разработанное на Kotlin.
   Ключевые особенности:
   - Поддержка 10 направлений перевода: секунды, минуты, часы, дни, недели и др.
   - Интуитивный интерфейс с мгновенной конвертацией.
   - Оптимизированная работа с различными единицами времени.

3. MetroSPB-Navigator
   Роль: C# / Desktop Application
   Описание: Приложение для поиска кратчайшего пути в метрополитене Санкт-Петербурга.
   Ключевые особенности:
   - Использует алгоритм Дейкстры для расчета оптимального маршрута.
   - Учитывает время в пути и пересадки между станциями.
   - Визуализация маршрута и расчет времени поездки.

4. KrasnodarMapApp
   Роль: Android (Yandex MapKit)
   Описание: Приложение геолокации для Краснодара с расширенной картографией и навигацией.
   Ключевые особенности:
   - Интеграция Яндекс Карты API с кастомными маркерами.
   - Реализация навигации и геолокационных сервисов.
   - Оптимизация рендеринга карты для плавной работы.

5. AirHockey (Аэрохоккей)
   Роль: C++ / Game Development
   Описание: Захватывающая 3D-игра для ПК, где скорость, стратегия и реакция играют ключевую роль.
   Ключевые особенности:
   - Реализация физического движка с обработкой столкновений.
   - Оптимизированный Game Loop для плавного геймплея.
   - Мультиплеер для игры с друзьями.

6. TeaShop Simulator
   Роль: C# / Desktop WinForms
   Описание: Десктопное приложение, имитирующее работу чайного магазина.

   - Реалистичная симуляция процесса покупок.
